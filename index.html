SOBRE A NOSSA BIBLIOTECA

Nossa biblioteca oferece um ambiente moderno, confortável e totalmente preparado para apoiar os estudos dos alunos. Contamos com um acervo diversificado, incluindo livros acadêmicos, literatura, periódicos, materiais digitais e recursos multimídia. Nosso espaço foi projetado para estimular a concentração, a criatividade e o desenvolvimento acadêmico.

Além disso, disponibilizamos salas de estudo individuais e em grupo, computadores modernos, acesso à internet de alta velocidade e suporte especializado. Nosso objetivo é proporcionar um ambiente completo para pesquisa, aprendizado e crescimento intelectual, sempre alinhado às necessidades dos estudantes.

AS UNIDADES

Unidade 1 — Python

Apresenta os fundamentos da linguagem Python, abordando sintaxe básica, estruturas de controle, manipulação de dados e primeiros passos na lógica de programação. Introduz também aplicações práticas que ajudam o aluno a desenvolver raciocínio computacional.

Unidade 2 — Docker

Explora os conceitos essenciais de containers, mostrando como o Docker facilita a criação, distribuição e execução de aplicações em ambientes isolados. Aborda imagens, containers, volumes e boas práticas para desenvolvimento moderno.

Unidade 3 — HTML

Explica a estrutura fundamental de páginas web, apresentando tags, organização de conteúdo, hierarquia de elementos e construção de layouts simples. Fornece a base necessária para iniciar no desenvolvimento front‑end.
